# Austo-Automobile-case-study
This Python-based case study analyzes **Austo Automobile** business data to identify sales trends, customer demand patterns, and operational inefficiencies. Using data cleaning, exploratory analysis, and visualization, the project delivers actionable insights to support inventory planning, revenue growth, and data-driven strategic decision-making.

# **✤ Background**

Austo Motor Company is a leading automobile manufacturer specializing in SUV, Sedan, and Hatchback segments. During a recent board meeting, concerns were raised regarding the effectiveness of the current marketing campaigns. To address these concerns, the company engaged an analytics professional to derive data-driven insights and improve campaign efficiency.

# **✤ Objective**

To analyze customer demographic and financial data to understand purchasing behavior and demand patterns, enabling Austo Motor Company to optimize marketing strategies and enhance customer experience.

# **✤ Dataset Overview**

| Feature          | Description                                       |
| ---------------- | ------------------------------------------------- |
| Age              | Age of the customer (in years)                    |
| Gender           | Gender of the customer (Male/Female)              |
| Profession       | Customer’s occupation                             |
| Marital_status   | Married or Single                                 |
| Education        | Graduate or Post Graduate                         |
| No_of_Dependents | Number of financial dependents                    |
| Personal_loan    | Whether the customer has a personal loan (Yes/No) |
| House_loan       | Whether the customer has a housing loan (Yes/No)  |
| Partner_working  | Employment status of partner (Yes/No)             |
| Salary           | Customer’s salary                                 |
| Partner_salary   | Partner’s salary (if applicable)                  |
| Total_salary     | Combined household income                         |
| Price            | Price of the automobile                           |
| Make             | Automobile type (SUV, Sedan, Hatchback)           |

# **✤ Key Business Questions Addressed**

Is there a gender preference when purchasing SUVs?

Are salaried individuals more likely to purchase sedans?

Is there statistical evidence supporting Sheldon’s claim that salaried males prefer SUVs?

How does automobile spending differ by gender?

Do individuals with personal loans spend less on vehicles?

Does having a working partner influence the purchase of high-priced cars?

# **✤ Approach & Analysis**

Data cleaning and preprocessing

Exploratory Data Analysis (EDA)

Income and loan impact analysis

Gender-based and profession-based segmentation

Visualization of spending and demand trends

# **✤ Tools & Technologies**

Python

Pandas

NumPy

# **✤ Outcome**
The analysis provides actionable insights into customer preferences, income influence, and loan impact on automobile purchases, enabling Austo Motor Company to refine its marketing campaigns and target high-value customer segments more effectively.
